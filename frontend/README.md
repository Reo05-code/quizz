# Quiz App Frontend

React + TypeScript + Vite で構築されたフロントエンドアプリケーションです。

詳細なセットアップ手順や使い方は [README.md](../README.md) をご覧ください。

## 開発サーバーの起動

```bash
# Dockerを使う場合（推奨）
docker-compose up frontend

# ローカルで直接起動する場合
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## コード品質

```bash
# ESLint チェック
npm run lint

# ESLint 自動修正
npm run lint:fix

# Prettier でフォーマット
npm run format

# TypeScript 型チェック
npm run type-check
```
